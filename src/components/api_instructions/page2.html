https://StringToneJS.com/api/-=YOUR_TEXT=-
This is StringTone's API Endpoint

The most basic use of the utility is as simple as making a GET or POST request to the endpoint with the text you you want to convert substitued for "--YOUR_TEXT--". That's it! In goes the word, out comes its color! Click the endpoint's url above. You should see a new window open containing the text "#c760db", which is the color signature of
"-=YOUR_TEXT=-" (plus a lovely shade of purple).
